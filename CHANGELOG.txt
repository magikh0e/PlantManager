v1.6.0 — 2026-03-08
--------------------
FEATURE: Notification Reminders

  Browser push notifications for overdue tasks, environment alerts,
  and feeding reminders — fires while the app tab is open.

  Notification Engine
  - Bell icon in header opens notification settings modal
  - Red badge on bell shows count of active alerts
  - Check engine runs every 5 minutes plus 10s after page load
  - Deduplication suppresses repeat alerts for 4 hours
  - Notifications only fire when the tab is in the background
  - Clicking a notification brings the app tab to focus

  Task Reminders
  - Overdue tasks: notifies when nextDueDate is past today
  - Due today tasks: notifies when nextDueDate matches today
  - Both toggleable independently in settings

  Environment Threshold Alerts
  - New optional threshold fields on environments: Temp Min/Max,
    Humidity Min/Max, pH Min/Max
  - Fires immediately when a new reading exceeds a threshold
  - Also checked on the periodic 5-minute interval

  Feeding Reminders
  - Global "remind if no feeding in X days" setting (0 = disabled)
  - Checks each mother plant's most recent feeding timestamp
  - Alerts for plants that have never been fed (if older than interval)

  Settings
  - Master enable/disable toggle with browser permission status
  - Request Permission button when not yet granted
  - Sub-toggles for each notification category
  - Configurable feeding reminder interval in days


v1.5.0 — 2026-03-08
--------------------
FEATURE: Clone Lineage / Family Tree

  Visual SVG family tree tracing genetic lineage across generations.

  Lineage Tracking
  - New "Parent Mother" dropdown on plant form (visible when Source=Clone)
  - Links clone-origin mothers back to the mother they were cloned from
  - Generation tracking (Gen 0=seed, Gen 1=clone of seed, etc.)

  Family Tree Visualization
  - Pure SVG tree renderer in Analytics tab (full-width chart card)
  - Nodes show strain name, health color bar, generation, source icon
  - Dashed connectors between parent and child with clone count badges
  - Filter dropdown to focus on a specific root mother's lineage
  - "Full View" button opens large modal for detailed inspection
  - Archived mothers shown at reduced opacity for complete lineage

  Plant Integration
  - "Lineage" button on plant cards opens tree modal for that plant
  - Auto-navigates to the root ancestor's full lineage tree
  - Clicking any tree node opens that plant's edit form


v1.4.2 — 2026-03-08
--------------------
IMPROVEMENTS: Export Security & WhatsNew

  Export Integrity
  - Data exports now include an FNV-1a 32-bit checksum in a signed envelope
  - Imports verify checksum — tampered files are rejected with a clear error
  - Export envelope includes origin field (tracker.tropicalrootsmaui.com)
  - Imports are restricted to files exported from the official tracker origin
  - Legacy exports without an envelope are no longer accepted

  WhatsNew Modal
  - Added "View full changelog" link opening the GitHub changelog in a new tab

v1.3.0 — 2026-03-07
--------------------
FEATURE: Strain Library & Analytics Charts

  Strain Library
  - New "Strains" tab to build a personal genetics database
  - Track strain name, type, breeder, genetics/lineage, flowering time,
    difficulty level, THC/CBD ranges, terpenes, effects, flavors, yield
  - Photo upload for each strain entry

  - Clone Success Rate: bar chart comparing rates across mothers
  - Plant Health Distribution: donut chart of health statuses
  - IPM Overview: issues by type and severity (stacked bar chart)
  - Activity Timeline: GitHub-style heatmap of recent activity frequency
  - All charts are pure inline SVG — no external libraries
  - Responsive viewBox-based rendering, themed with CSS custom properties
  - Date range filter (7/30/90 days or all), environment and plant filters

  Dashboard
  - New "Strains" stat card with library count
  - New "+ Add Strain" quick action button

  Data
  - New "strains" collection in localStorage with automatic migration
  - Backward compatible — existing data is unaffected


v1.2.0 — 2026-03-07
--------------------
FEATURE: Feeding Log, Photo Journal, IPM Tracking

  Three new high-value features for comprehensive cultivation management.

  Feeding / Nutrient Log
  - New "Feeding" tab to track nutrient feedings per plant or environment
  - Log nutrient name, amount, unit, water volume, pH in/out, EC/PPM in/out
  - pH drift highlighted when runoff deviates more than 0.5
  - Filter feedings by plant or environment
  - Plant tags on each feeding card for quick identification

  Photo Journal
  - New "Journal" tab with visual photo grid timeline
  - Upload photos linked to a specific plant and growth stage
  - Click any photo card to view full-size detail with metadata
  - Filter by plant; photos auto-resized to save localStorage space
  - Delete photos from the detail view

  IPM Log (Pest & Disease Tracking)
  - New "IPM" tab to log pest sightings, diseases, and deficiencies
  - 11 common pest/problem presets (aphids, spider mites, powdery mildew, etc.)
  - Severity badges: low (green), medium (gold), high (orange), critical (red)
  - Track treatments and mark issues as resolved
  - Filter by plant, severity, and active/resolved status
  - Edit existing issues and update treatments

  Integration
  - Plant cards now include + Feed, + Photo, + IPM quick action buttons
  - Dashboard: new "Active Issues" stat card (orange when issues are open)
  - Dashboard: three new quick actions (+ Log Feeding, + Add Photo, + Log Issue)
  - Existing "Environ" tab renamed to "Readings" for clarity
  - All actions logged in the activity feed

v1.1.0 — 2026-03-07

FEATURE: Grow Environments

  Added grow tents and environments as first-class entities. Plants can
  now be assigned to an environment, and environmental readings (temp,
  humidity, pH, EC/PPM, light hours) are tracked per environment instead
  of per plant.

  - New "Environs" tab with full CRUD (add, edit, search, archive)
  - Environment cards display type badge, dimensions, light setup,
    assigned plant count, and latest sensor readings
  - Six environment types: tent, room, greenhouse, outdoor, closet, other
  - Plants module: replaced free-text location field with environment
    dropdown selector
  - Environment readings tab: filter readings by specific environment
  - "+ Log Reading" shortcut on each environment card (pre-selects env)
  - Dashboard: new Environments stat card and "+ Add Environ" quick action
  - Backward compatible — existing plants with old location field and
    legacy per-plant readings still display correctly


v1.0.0 — 2026-03-07
--------------------
Initial release of Tropical Roots Maui - Plant Manager.

  CORE
  - Single self-contained HTML file, no server required
  - All data persisted in browser localStorage
  - Light / dark theme toggle with system preference detection
  - Full data import / export (JSON backup & restore)
  - Responsive layout for mobile, tablet, and desktop

  MOTHER PLANTS
  - Add, edit, search, and archive mother plants
  - Track strain, genetics, source, acquisition date, growth stage
  - Photo upload with automatic canvas resize (400px, JPEG)
  - Health status badges: thriving, healthy, stressed, sick, quarantine
  - Clone success rate tracking per mother
  - Filter plants by status

  CLONE TRACKING
  - Log clone batches linked to a mother plant
  - Record cut date, quantity, medium, rooting hormone, and notes
  - Auto-calculated success rate feeds back to mother card
  - Search and filter clone history

  TASK SCHEDULING
  - Create one-time or recurring care tasks (daily, weekly, biweekly,
    monthly, quarterly)
  - Assign tasks to specific plants
  - Mark tasks complete with automatic next-due advancement
  - Priority levels: low, medium, high, urgent
  - Overdue and due-today highlighting
  - Filter by status and plant

  ENVIRONMENT READINGS
  - Log temperature, humidity, light hours, pH, and EC/PPM
  - Latest-reading summary cards with current values
  - Trend history table with SVG sparkline mini-charts
  - Filter readings by plant (v1.0) or environment (v1.1)

  DASHBOARD
  - At-a-glance stat cards: mothers, clones, environments, tasks due,
    health overview
  - Quick action buttons for all major workflows
  - Chronological activity feed (recent adds, edits, completions)
  - Hero banner with Kanehiwa alchemical sanctuary artwork

  BRANDING & SEO
  - Tropical Roots Maui branding with TRM logo
  - Full Open Graph and Twitter Card meta tags
  - Canonical URL, favicon references, structured keywords
