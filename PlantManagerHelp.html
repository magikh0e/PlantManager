Getting Started
⚡

First Steps

Plant Manager is an offline-first, single-page app that stores all data in your browser's localStorage. No account is required to get started. Simply open tracker.tropicalrootsmaui.com and begin adding plants.

The recommended first steps are:

  • Add an Environment — create your grow tent, room, or outdoor space on the Environs tab.
  • Add a Mother Plant — go to the Plants tab and tap + Add Mother. Pick a strain from the catalog or type one in.
  • Set up Tasks — schedule watering, feeding, or pruning reminders so nothing slips through the cracks.
  • Export your data — once you have entries, use the gear menu to export a JSON backup file.
Tip All data lives in your browser. Clearing browser data or switching browsers will remove it unless you export a backup or enable Cloud Sync.

Dashboard
▦

Dashboard Overview

The Dashboard is your home screen. It provides a bird's-eye view of your entire garden.

Stat Cards

The top row shows live stat cards: Mother Plants, Flowering Count, Active Clones, Environments, Tasks Due, Health Overview, Active IPM Issues, Environment Alerts, Strains, Total Yield, plus Weather and Moon Phase widgets.

Rotating Tips

A tip banner cycles through 53 helpful tips. Use the arrow buttons to navigate tips manually, or let them auto-rotate every 30 seconds.

Quick Actions

29 quick action buttons let you jump directly to any feature without switching tabs: Add Mother, Add Environ, Log Clones, Add Task, Log Reading, Log Feeding, Add Photo, Log Issue, Add Strain, Log Harvest, KNF Calculator, Seed Bank, Quick Water, Daily Snapshot, Feed Schedule, Costs, Grow Cycles, Compare Plants, Soil Mix Calc, Export Report, Pest ID, Pest Guide, VPD Calc, DLI Calc, Deficiency Wizard, Grow Runs, Breeding, Calibration, and Room Layout.

Overdue Tasks & Environment Alerts

If any tasks are overdue, they appear at the top of the dashboard with a red warning indicator. You can check them off right from the dashboard. Environment alerts display when any reading exceeds the thresholds you set (temp, humidity, pH).

Activity Feed

The bottom of the dashboard shows your 15 most recent actions (adding plants, logging feedings, completing tasks, etc.) with relative timestamps.

Plants
🌿

Mother Plants

The Plants tab is where you manage your mother plants. Each plant has a card showing its photo, health status badge, strain name, type, breeder, age, environment, growth stage, flowering dates, and clone success rate.

Adding a Plant

Tap + Add Mother and fill in the form. You can pick a strain from your Strain Library (which auto-fills strain, type, and breeder) or enter details manually. Fields include: strain name, strain type (indica/sativa/hybrid), breeder, genetics, source (seed/clone/tissue culture), date acquired, environment, growth stage, health status, photo, and notes.

Health Status

Each plant has a health status: Thriving, Healthy, Stressed, Sick, or Quarantine. This status appears as a colored badge on the card and is used for filtering.

Plant Card Actions

Each card has action buttons along the bottom:

  • Edit — modify any field. If you change the health status, a health check is auto-logged.
  • + Clone — log a clone batch from this mother.
  • + Task — create a task linked to this plant.
  • + Feed — log a feeding entry.
  • + Photo — add a journal photo.
  • + IPM — log a pest or disease issue.
  • 🔎 Identify — open the Pest Identification tool to snap a photo and identify pests or diseases.
  • + Harvest — record a harvest.
  • ❤ Health — log a 1–5 health score (see Health Check below).
  • 🏷 Tag — generate a printable QR code tag for this plant.
  • 📅 Timeline — view the full chronological event history combining feedings, photos, IPM, harvests, health checks, dry-backs, clones, and task completions.
  • ⚖ Dry-Back — log dry-back weight readings and view a line chart of weight over time.
  • 🌳 Lineage — view the full clone family tree visualization.
  • 📄 Report — generate a printable Plant Report with full grow history, stats, feedings, harvests, IPM log, and health checks.
  • 📈 Compare — select this plant for side-by-side comparison with 1–2 other plants.
  • 🔬 Trichome — log trichome readings (clear/cloudy/amber percentages) with harvest-window recommendations.
  • ✂ Training — log training events (topping, LST, SCROG, etc.) with technique badges and notes.
  • 🧬 Breed — log pollen collection, crosses, or seed harvests for this plant.
  • Archive — soft-delete the plant (can be restored via import).
Search & Filter

Use the search bar to find plants by strain name, breeder, environment, or notes. The dropdown filter narrows results by health status.

Health Check Dots

Colored dots appear below each plant's name showing recent health scores at a glance: green (4–5), gold (3), orange (2), and red (1).

Batch Operations (Select Mode)

Tap Select to enter batch mode. Check multiple plants, then use the batch bar to:

  • Update Health — set the same health status on all selected plants.
  • Update Stage — change growth stage for all selected.
  • Move Environ — reassign all selected to a different environment.
  • Print Tags — generate QR tags for all selected plants.
  • Archive Selected — archive multiple plants at once.
Tip The Flowering field on plant cards tracks flowering start/end dates and auto-calculates days in flower. Fill in the flowering start date when you flip to 12/12. If the strain has a flowering time set, plant cards will show a harvest estimate badge with a progress bar and estimated harvest window.
Environments
⛺

Grow Environments

Environments represent your physical grow spaces — tents, rooms, greenhouses, or outdoor areas.

Environment Fields
  • Name — a descriptive name like "Veg Tent 4x4" or "Flower Room".
  • Type — Tent, Room, Greenhouse, Outdoor, Closet, or Other.
  • Dimensions — e.g., 4x4x6.5.
  • Light Type & Schedule — e.g., LED 600W, 18/6.
  • Alert Thresholds — set min/max for temperature, humidity, and pH. When readings exceed these limits, the Dashboard will show environment alerts.
Environment Cards

Each card shows the environment type badge, name, dimensions, light info, plant count, and the latest reading values (temp, humidity, pH). Values that exceed your thresholds appear in red. A pH/EC calibration badge shows days since last calibration (with a warning color if overdue).

Environment Card Actions

Each environment card has: Edit, + Log Reading, 🌡 VPD (open VPD Calculator pre-filled with this environment’s latest reading), 📐 Layout (open the Room Diagram for this environment), and Archive.

Clones
✂

Clone Tracking

The Clones tab tracks clone batches taken from your mother plants. Each batch records how many cuttings were taken, how many rooted successfully, and how many failed.

Clone Batch Fields
  • Mother Plant — the source mother.
  • Date Taken — when cuttings were made.
  • Total Taken / Rooted / Failed — numeric counts.
  • Rooting Status — Rooting, Rooted, Transplanted, or Failed.
  • Medium — Rockwool, Rapid Rooter, Soil, Perlite, Aero, or Water.
  • Notes — free-text observations.
Stats & Progress

The top of the tab shows aggregate stats: total taken, total rooted, total failed, total batches, and overall success rate. Each batch card shows a color-coded progress bar (green = rooted, red = failed, gray = pending).

Clone success rates automatically appear on the corresponding mother plant cards on the Plants tab.

Tasks
☑

Task Scheduler

Create tasks to remind yourself of watering, feeding, pruning, IPM treatments, repotting, or cleaning. Tasks can be linked to a specific plant or set as global (all plants).

Task Types

Watering, Feeding, Pruning, IPM, Repotting, Cleaning, and Other. Each type has a unique icon for easy scanning.

One-Time vs Recurring

One-time tasks disappear after completion. Recurring tasks automatically re-schedule at your chosen interval (e.g., every 3 days). Completing a recurring task advances the due date to the next interval.

Task Sections

Tasks are organized into sections: Overdue (red), Due Today (gold), Upcoming (default), and Recently Completed (faded). Tap the circular checkbox to complete a task.

Filters

Filter by plant or by task type to focus on what matters.

Tip Overdue tasks also appear on the Dashboard. Completing them from either location works the same way.
Feeding
🧪

Feeding Log

Log every feeding and watering event with detailed nutrient tracking.

Feeding Entry Fields
  • Plant & Environment — link the feeding to a plant and environment.
  • Products — add one or more nutrient products with amounts and units. The built-in catalog includes 75+ products from major brands: General Hydroponics, Fox Farm, Advanced Nutrients, Botanicare, Canna, Down To Earth, Athena, Gaia Green, Neptune's Harvest, and KNF inputs.
  • Volume & Unit — total solution volume in gallons or liters.
  • pH In / pH Out — track pH drift. The card highlights drift greater than 0.5.
  • EC/PPM — electrical conductivity measurement.
  • Water Source — Tap, Filtered, Rain, RO, or Well.
  • Notes — any observations.
Quick Water

The Quick Water button (on the Feeding toolbar and Dashboard) opens a streamlined form for plain water events. It captures plant, environment, volume, pH in/out, water source, and notes — skipping the full nutrient product selector.

Custom Products

If a nutrient isn't in the catalog, tap + Add Custom Product to save your own. Custom products persist in your data and appear in the product picker going forward.

Photo Journal
📸

Photo Journal

Build a visual timeline of each plant's growth. Upload photos with growth stage tags and notes. Photos are stored locally as compressed base64 images (auto-resized to 400px max dimension).

Adding a Photo

Tap + Add Photo, select a plant, upload an image, choose the growth stage (Seedling, Vegetative, Flowering, Harvest, Curing), and add optional notes.

Photo Grid

Photos display in a grid with the plant name, date, and growth stage. Tap any photo to view full-size with details, or delete it.

Filter

Use the plant dropdown to filter photos to a single plant.

IPM — Pest & Disease
🛡

IPM Log

The IPM (Integrated Pest Management) tab tracks pest sightings, disease outbreaks, and treatment applications.

Issue Fields
  • Pest/Disease — choose from 11 common types: aphids, spider mites, fungus gnats, whiteflies, thrips, powdery mildew, bud rot, root rot, nutrient burn, nutrient deficiency, or other.
  • Severity — Low, Medium, High, or Critical.
  • Plant & Environment — link to the affected plant and/or environment.
  • Treatment Applied — pick from a catalog of 60+ IPM products organized by category: insecticides, miticides, fungicides, biologicals, organic remedies, KNF treatments, and environmental controls.
  • Photo & Notes — document what you observe.
Inline Pest Identification

The IPM form includes a built-in Pest Identification section with 4 buttons: Snap Photo (camera/upload), Auto-ID (Google Cloud Vision AI — requires API key), Lens (opens Google Lens for manual lookup), and Guide (opens the Visual Pest Guide). When Auto-ID identifies a pest, tap Apply to auto-fill the pest dropdown, issue type, and severity without leaving the form.

Active vs Resolved

Issues start as Active and can be marked Resolved when the problem is handled. Use the status filter to view only active or resolved issues.

Dashboard Integration

Active IPM issues show as a count on the Dashboard stat cards. The number helps you track unresolved pest pressure at a glance.

Harvest
🌾

Harvest Log

Record harvest data for each plant including weights, cure tracking, and quality ratings.

Harvest Fields
  • Plant & Environment — which plant and where it grew.
  • Harvest Date and Flowering Start/End — calculates total flowering days.
  • Wet Weight & Dry Weight (g) — auto-calculates dry ratio percentage.
  • Trim Type — Wet, Dry, or Both.
  • Cure Method — Jar, Bag, Grove Bag, Bucket, Hang Dry, or Other.
  • Cure Status — Fresh, Drying, Curing, or Complete.
  • Quality Rating — 1–5 stars.
  • Terpene Notes, Flavor Notes, Storage Location — detailed record keeping.
  • Photo & Notes — document the final product.
Yield Tracking

Total yield appears on the Dashboard in grams or ounces (auto-converts at 28.35g/oz). Harvest data feeds into the Analytics tab's yield charts.

Strains & Catalog
🧬

Strain Library

The Strains tab is your personal strain library plus a built-in catalog of 360+ strains from Maui-based breeders.

Your Strains

Manually add strains or import them from the catalog. Each strain card shows name, type (indica/sativa/hybrid), breeder, genetics, flowering time, and average review rating if you have reviews.

Browse Catalog

The built-in catalog includes strains from 7 breeders: Sticky Finger Seeds, Hawaiian Budline, Ziplock Seeds, Makena Genetics, Deep Green Genetics, Aloha Island Genetics, and CloneFreak (clones only). Filter by breeder, search by name, and quick-add strains to your library.

Seed Bank

Access the Seed Bank from the Strains toolbar or the Dashboard. Track seed packets with strain name, breeder, count, type (feminized/regular/autoflower), source, purchase date, germination rate, and notes. The Browse Catalog button inside the Seed Bank lets you quick-add seed entries from the strain catalog (excludes clone-only providers).

Strain Reviews

Tap the ⭐ Reviews button on any strain card to write a personal grow review. Reviews capture:

  • Overall rating (1–5 stars)
  • Grow difficulty, yield rating, and actual flowering time
  • Grow method, effects, terpene/flavor notes
  • Grow tips and general notes
  • Optional photo

Average ratings display directly on strain cards so you can compare strains at a glance.

Tip When adding a plant on the Plants tab, the strain picker auto-fills type and breeder from your library. Build your strain library first for faster plant entry.

Analytics Charts

The Analytics tab renders 14 interactive SVG charts from your data. All charts are pure SVG — no external libraries required.

Filters

Use the date range (7 days, 30 days, 90 days, All Time), environment, and plant dropdowns to narrow chart data.

Charts
  • Environment Trends — line chart of temperature, humidity, pH, and EC/PPM over time.
  • Feeding pH Drift — line chart comparing pH In vs pH Out across feedings.
  • Clone Success Rates — bar chart comparing rooted vs failed counts per mother.
  • Family Tree — visual lineage tree showing mother-clone relationships.
  • Health Distribution — donut chart of plant health statuses.
  • IPM Overview — bar chart of pest/disease types by severity.
  • Activity Heatmap — calendar heatmap showing daily activity levels.
  • Harvest Yield Timeline — line chart of individual and cumulative dry weight over time.
  • Yield by Strain — bar chart ranking total yield per strain (top 12).
  • Quality Distribution — donut chart of harvest star ratings with average score.
  • Flowering Days vs Yield — bar chart correlating flowering duration with dry weight.
  • Cost Breakdown — donut chart showing expense distribution across 7 categories with dollar totals and a color-coded legend.
  • VPD Trends — line chart tracking vapor pressure deficit over time with zone bands for each growth stage.
  • Cost per Gram by Strain — bar chart showing dollar cost per gram of dried flower, allocated by strain (direct + proportional shared expenses).
Calendar
📅

Grow Calendar

A month-view calendar that overlays all your events: tasks, feedings, journal photos, harvests, IPM entries, and harvest estimate windows. Each day cell shows colored dots for event types and the moon phase icon.

Navigation

Use the left/right arrows to navigate months, or tap Today to jump back to the current month.

Day Detail

Tap any day to see a full list of events for that date, plus the moon phase name and KNF planting guidance for that day.

Color Legend
  • Cyan dot — Tasks
  • Green dot — Feedings
  • Gold dot — Journal entries
  • Orange dot — Harvests
  • Red dot — IPM issues

Reading Fields

Latest Readings

Tools & Extras
🔧

Additional Tools

KNF Input Calculator

Open from the Dashboard or gear menu. Enter your water volume, select a KNF input (FPJ, FFJ, FAA, OHN, LABS, IMO, WCA, WCA-P, BRV, or Molasses), and toggle between Foliar Spray (1:1000) and Soil Drench (1:500) dilution rates. The calculator shows the exact amount needed.

Moon Phase Tracker

The moon widget on the Dashboard shows the current phase, illumination, and time until the next phase. Tap it for a detailed modal with KNF planting guidance for each of the 8 lunar phases, upcoming phase dates, and a visual cycle overview.

QR Code Plant Tags

Generate printable QR code labels for any plant. Each tag includes a QR code (encoding strain, breeder, stage, and date), the plant name, breeder, type, stage, environment, date acquired, and health status. Print single tags from a plant card or batch-print all tags from the gear menu or select mode.

Daily Snapshot

Open from the Dashboard to generate a full daily report. The snapshot includes the date, current weather, moon phase with KNF guidance, pending tasks, active IPM issues, environment alerts, and recent feedings. You can copy to clipboard for pasting elsewhere or save to journal as a text note.

Health Check / Score Tracker

Tap ❤ Health on any plant card to log a quick 1–5 score with optional notes. Scores are: 1 = Critical, 2 = Poor, 3 = Fair, 4 = Good, 5 = Excellent. Recent scores appear as colored dots on plant cards, and health changes during plant edits are automatically logged.

Lineage Tree

Tap 🌳 Lineage on a plant card to view a visual family tree showing the mother plant and all clone batches branching from it. The tree is rendered as an interactive SVG on the Analytics tab.

Plant Timeline

Tap 📅 Timeline on any plant card to view a chronological event history combining 8 data sources: feedings, photos, IPM logs, harvests, health checks, dry-back readings, clone batches, and task completions. Events are sorted by date with color-coded dots for each type.

Feed Schedule Templates

Open from the Dashboard or gear menu. Choose from 4 built-in nutrient schedules: GH Flora Series (10 weeks), Fox Farm Trio (8 weeks), Gaia Green Organic (5 weeks), or KNF Inputs (8 weeks). Preview the weekly schedule, then apply it to a plant to auto-create one-time feeding tasks for each week.

Dry-Back Tracking

Tap ⚖ Dry-Back on a plant card to log weight readings over time. Enter the weight, unit (g/oz/lb), and optional notes. A line chart tracks the dry-back curve and a history table lets you review or delete past entries.

Harvest Estimate

If a plant's strain has a flowering time set and the plant has a flowering start date, a harvest estimate badge appears on the plant card. It shows a progress bar, estimated harvest window, and days remaining. Harvest estimate events also appear on the Calendar.

Cost Tracker

Open from the Dashboard or gear menu. Log grow expenses across 7 categories: Nutrients, Equipment, Soil/Media, Seeds/Clones, IPM, Utilities, and Other. The summary shows total spent, spending this month, and average per month. A Cost Breakdown donut chart appears on the Analytics tab.

Grow Cycle Templates

Open from the Dashboard, gear menu, or plant cards. Choose from 4 built-in grow cycle templates: Indoor Photoperiod (16 weeks / 9 phases), Indoor Autoflower (12 weeks / 7 phases), Outdoor Tropical Maui (21 weeks / 6 phases), or Clone to Flower (12 weeks / 7 phases). Each phase includes week range, phase name, light schedule, and a list of tasks. Preview all phases in a table, then apply to a plant — this auto-creates one-time tasks with calculated due dates from seedling through harvest.

Soil Mix Calculator

Open from the Dashboard or gear menu. Choose from 4 pre-built soil recipes: Coots Mix (classic 1:1:1 living soil), BuildASoil 3.0, Simple Coco Mix, or Tropical Living Soil (Maui-style). You can also start a Custom Mix from scratch. The calculator has 14 components across 3 categories: Base Mix (Peat Moss, Coco Coir, Compost), Aeration (Perlite, Pumice, Rice Hulls), and Amendments (Worm Castings, Bone Meal, Blood Meal, Kelp Meal, Gypsum, Dolomite Lime, Rock Phosphate, Azomite). Adjust the total volume and unit (gallons or liters), edit any ratio or amount, and calculated quantities update live. Save custom recipes for later use.

Plant Comparison

Open from the Dashboard or a plant card's 📈 Compare button. Select 2–3 plants with checkboxes, then tap Compare. A comparison table shows 11 metrics side-by-side: strain, type, age, stage, health, environment, feedings, IPM issues, clone rate, total yield, and average health score. Best values are highlighted in cyan. Mini bar charts below the table compare feedings, yield, and health scores visually.

Export Report (Print-to-PDF)

Open from the Dashboard, gear menu, or a plant card's 📄 Report button. Choose between two report modes:

  • Plant Report — full grow history for a single plant including strain info, stat grid (8 metrics), feeding history (last 20), harvests, IPM log, and health checks.
  • Garden Report — overall summary of your entire garden including plant count, environments, flowering count, total yield, total expenses, cost per gram, plant roster, environment list, harvest summary, and expense breakdown by category.

Reports open in a new browser window with a dark theme on screen and clean black-and-white formatting for printing. Tap Print / Save PDF to open your browser's print dialog and save as PDF.

Pest Identification Tool

Open from the Dashboard (🔎 Pest ID), gear menu (🔎 Pest Identifier), or a plant card's 🔎 Identify button. The tool offers three identification methods:

  • Google Cloud Vision AI — snap or upload a photo and the app sends it to the Vision API for automatic analysis. It uses LABEL_DETECTION and WEB_DETECTION to match against 10 known cannabis pests and diseases, displaying the result with a confidence percentage. Tap Log as IPM Issue to pre-fill the IPM form with the identified pest, issue type, and suggested severity. Requires a Vision API key (free tier: 1,000 images/month).
  • Google Lens — opens Google Lens in a new tab for manual visual identification. No API key required.
  • Visual Pest Guide — a built-in reference with detailed cards for all 10 supported pests and diseases. Each card shows an emoji icon, symptom descriptions, and recommended treatments.
Visual Pest Guide

Open from the Dashboard (📚 Pest Guide), the Pest ID tool, or the IPM form. The guide covers 10 common cannabis issues: Aphids, Spider Mites, Fungus Gnats, Whiteflies, Thrips, Powdery Mildew, Bud Rot, Root Rot, Nutrient Burn, and Nutrient Deficiency. Each card includes visual identification signs and treatment options.

Pest ID Settings

Open from the gear menu (🤖 Pest ID Settings) or the link in the Pest ID tool. Enter your Google Cloud Vision API key — get one free at Google Cloud Console (enable the Vision API, then create a key). Use the Test API Key button to verify your key works before saving. Without a key, Google Lens and the Visual Pest Guide remain fully functional.

VPD Calculator

Open from the Dashboard, gear menu, or the 🌡 VPD button on environment cards. The calculator auto-fills temperature and humidity from the latest environment reading. Enter an optional leaf temperature offset (default: air temp − 2°F). The result shows your VPD in kPa with color-coded zone indicators for seedling (0.4–0.8), vegetative (0.8–1.2), early flower (1.0–1.5), and late flower (1.2–1.6). A VPD trend chart also appears on the Analytics tab.

DLI Calculator

Open from the Dashboard or gear menu. Select your light type (LED, HPS, CMH, Fluorescent, or Sunlight), enter wattage, and set photoperiod hours. The calculator estimates PPFD from light efficiency and computes Daily Light Integral (DLI) in mol/m²/day. Optimal ranges are shown for each growth stage: seedling (12–18), veg (18–30), and flower (30–45).

Trichome Harvest Tracker

Open from the 🔬 Trichome button on plant cards. Log trichome development readings with linked sliders for clear, cloudy, and amber percentages (always totaling 100%). The tracker provides harvest-window recommendations: “Too early” when clear > 50%, “Peak THC window” when cloudy ≥ 50% and amber < 20%, and “Balanced / couch-lock” when amber ≥ 20%. A stacked bar chart shows reading history, and a compact trichome badge appears on flowering plant cards.

Training Log

Open from the ✂ Training button on plant cards. Choose from 10 techniques: Topping, FIM, LST, HST, Supercrop, Mainline, SCROG, Lollipop, Defoliation, or Other. Add optional position info, a photo, and notes. Training entries appear on the Plant Timeline and Calendar.

Nutrient Deficiency Wizard

Open from the Dashboard, gear menu, or the 🩹 Deficiency Wizard button in the IPM form. The wizard walks you through 3 steps: (1) Where are symptoms? (old growth, new growth, all), (2) What do you see? (yellowing, browning, spots, curling, purple), (3) What pattern? (interveinal, edges, tips, uniform). Based on your answers, it identifies the most likely deficiency (Nitrogen, Magnesium, Iron, Potassium, Calcium, Phosphorus, Manganese, or Nutrient Burn) and provides fix instructions. A Log as IPM Issue button pre-fills the IPM form.

Grow Run Manager

Open from the Dashboard or gear menu. Create named grow runs, assign plants, and track start/end dates. Each run shows aggregate statistics: plant count, total yield, total cost, and average health. Mark runs as completed when finished. Use Compare Runs to select two runs and view side-by-side stats.

Breeding / Pollen Tracker

Open from the Dashboard, gear menu, or the 🧬 Breed button on plant cards. Log three entry types: Pollen Collection (date, storage method, viability notes), Cross (mother × father, method, notes), and Seed Harvest (parents, seed count, offspring strain name). Filter entries by plant or type.

pH/EC Calibration Log

Open from the gear menu (⚗ Calibration Log). Log meter calibrations with meter type (pH, EC, or both), buffer solutions used, and notes. Set a calibration interval (default: 30 days). A badge on environment cards shows days since last calibration and turns orange/red when overdue.

Fleet View / Table Mode

Toggle between Grid and Table view on the Plants tab using the ☰ Table / ▦ Grid button in the toolbar. Table mode shows a sortable table with 9 columns: Strain, Type, Stage, Health, Age, Environment, Last Fed, Days Since IPM, and Yield. Click any column header to sort ascending or descending.

Grow Room Diagram

Open from the 📐 Layout button on environment cards or the gear menu. Set grid dimensions (rows × columns) for your grow space. Click any cell to assign a plant from a picker, or click an occupied cell to unassign. Layouts save per environment.

Notifications

Enable browser notifications from the gear menu. The app will send alerts for overdue tasks and environment readings that exceed your thresholds.

Theme Toggle

Switch between dark mode (default — dark green/black) and light mode (cream/white) using the moon icon in the header.

Data & Backup
💾

Data Management

Local Storage

All data is stored in your browser's localStorage under the key _trm_pm. This means data persists between sessions but is tied to the specific browser and device you use.

Export

Go to gear menu → Export Data to download a complete JSON backup. The file includes a checksum for integrity verification. Export regularly — it only takes a second and protects all your grow records.

Import

Go to gear menu → Import Data to restore from a JSON backup. This replaces all current data with the imported file. The import validates the checksum to ensure the file hasn't been corrupted.

Cloud Sync (Firebase)

Tap the ☁ Cloud Sync button in the header to connect with Firebase. Cloud sync provides automatic backup and cross-device synchronization. Your data is encrypted and stored securely. Log in with your email to enable sync.

Tip Export before clearing browser data! Clearing cookies, site data, or cache will permanently delete your Plant Manager data unless you have a backup or Cloud Sync enabled.

PWA Install

Plant Manager is a Progressive Web App (PWA). You can install it on your phone or computer for a native app-like experience:

  • iOS Safari — tap the Share button → "Add to Home Screen".
  • Android Chrome — tap the three-dot menu → "Install app" or "Add to Home Screen".
  • Desktop Chrome/Edge — click the install icon in the address bar, or go to the three-dot menu → "Install Tropical Roots Maui - Plant Manager".

Once installed, the app opens in its own window without browser chrome and works offline thanks to the service worker cache.
